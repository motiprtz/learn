# 🚀 הפעלת GitHub Pages - מוטי פרץ

## ✅ מה שכבר נעשה:

1. ✅ הקוד עבר לחשבון של מוטי: https://github.com/motiprtz/modernism
2. ✅ הקבצים עודכנו לנתיב הנכון (`/modernism/`)
3. ✅ GitHub Actions מוגדר ומוכן

---

## 📋 מה מוטי צריך לעשות עכשיו:

### שלב 1: הפעל GitHub Pages

1. **היכנס לריפו**: https://github.com/motiprtz/modernism
2. **לחץ על Settings** (למעלה)
3. **בצד שמאל, לחץ על Pages**
4. תחת **"Build and deployment"**:
   - **Source**: Deploy from a branch
   - **Branch**: בחר **`gh-pages`** (אם אין עדיין, זה יופיע אחרי הפעם הראשונה)
   - **Folder**: **`/ (root)`**
5. **לחץ Save**

---

### שלב 2: המתן ל-Deployment

1. **עבור ל-Actions**: https://github.com/motiprtz/modernism/actions
2. תראה workflow בשם **"Deploy to GitHub Pages"**
3. המתן עד שיופיע ✅ **סימן ירוק** (לוקח 2-3 דקות)

---

### שלב 3: גש לאתר! 🎉

**האתר יהיה זמין ב:**

```
https://motiprtz.github.io/modernism/
```

---

## 🔧 אם יש בעיות:

### האתר לא נטען?

1. ✅ וודא ש-GitHub Pages מופעל (Settings → Pages)
2. ✅ וודא שבחרת branch **`gh-pages`** (לא `main`)
3. ✅ בדוק ב-Actions שה-workflow הסתיים בהצלחה
4. ✅ המתן 2-3 דקות ונקה cache (Ctrl+Shift+R)

### עדיין דף ריק?

אם אחרי 5 דקות עדיין דף ריק:
1. בדוק שה-Actions הסתיים בהצלחה
2. בדוק שב-Settings → Pages מופיע הכתובת הנכונה
3. נסה לפתוח בחלון פרטי (Incognito)
